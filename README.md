# Hasta Kala Shopping App

## Overview
Hasta Kala Shopping App is an Android-based e-commerce application developed to promote handmade and handicraft products through a modern and user-friendly mobile platform.

The application allows users to browse products, manage cart items, and explore shopping features with a clean and responsive interface.

---

## Features
- User-friendly UI
- Product browsing
- Shopping cart functionality
- Dashboard analytics
- Responsive Android application
- Modern Material Design interface

---

## Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- Firebase
- Material Design
- Gradle
- GitHub

---

## Problem Statement
Local artisans and handicraft sellers often struggle to reach a wider audience due to limited digital presence. This application helps bridge that gap by providing a digital platform to showcase and sell handmade products.
